# 
「Tagging helper」 - 「記事」「ウェブページ」にタグ候補を表示するプラグインです。
====

## v0.5.1
- 対応バージョン: MT6, MT7

### 概要
- MT7対応
  - 記事とウェブページについて、MT7で動くようにした。
  - コンテンツタイプには非対応。

### 詳細仕様
- ブログ記事編集画面、ウェブページ編集画面
  - タグ入力欄の下に「全ブログ」の「記事」「ウェブページ」に設定してあるタグが候補として表示される
  - 別ブログの記事のタグも候補に出てくる
  - コンテンツデータのタグは候補にならない（コンテンツタイプ非対応）

- コンテンツデータ編集画面
  - （コンテンツタイプ非対応）
